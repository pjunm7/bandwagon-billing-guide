## 续费价格会涨吗？

不涨。这是搬瓦工明确的续费政策：**续费价格和首次购买价格一致**。

如果你首次购买时用了优惠码，续费也按折扣价来。反过来，首次没用优惠码，续费也无法补上——所以建议第一次购买时就用上优惠码。

## 搬瓦工续费时间线——防止忘记导致数据丢失

这块要说清楚，因为踩坑代价不小，特别是用了几年的老机器如果因为没留意续费而被回收，数据就没了。

续费流程大概是这样：

1. **到期前 7 天**：系统自动生成续费账单，同时发邮件通知到你的注册邮箱
2. **账单生成后 7 天内**：这是付款窗口，VPS 照常运行
3. **7 天后还没付款**：VPS 被暂停（Suspend）
4. **暂停后账单仍然有效**：付了款就能恢复
5. **账单被取消（Terminate）**：VPS 和数据彻底没了，无法恢复

所以实际上有不少缓冲时间，不是一到期就没了。但最稳的还是提前充余额自动续费，省得操心。

## FAQ

**Q：搬瓦工的付款周期最长能选几年？**  
A：购买时最长只能选年付（1年）。没有2年付或3年付选项。想用多年，可以通过向账户预充多年份余额来实现自动续费。

**Q：余额充多了用不完怎么办？**  
A：账户余额可以用于后续续费或购买新套餐，不会过期。但如果申请退款，手续比较麻烦，所以建议按实际需要充。

**Q：续费时可以用优惠码打折吗？**  
A：目前搬瓦工已经取消了大部分常规优惠码。首次购买时如果用了有效优惠码，续费会自动享受该码的折扣。但首次没用的，续费时无法补充优惠码。

**Q：预充余额的钱是从哪里扣的？**  
A：系统会在账单生成时优先从账户余额扣款，无需手动操作，也不会主动扣你的支付宝或银行卡。

**Q：如果忘记续费，VPS 的数据会立刻没吗？**  
A：不会立刻没。账单生成后有 7 天付款窗口，窗口内 VPS 照常运行。超过 7 天后 VPS 会被暂停，但账单仍然有效，付款后可以恢复。只有账单被系统最终取消（Terminate）后，数据才会彻底删除。
